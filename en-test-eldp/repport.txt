Module Integration Report

We attempted two different methods to integrate our module (en-test-eldp) into IIAB.

Method 1: Direct Integration via Modules Directory
1. Placed the module directory en-test-eldp in:
   /library/www/html/modules
2. Added the module image en-test-eldp.jpg to:
   /library/www/html/js-menu/menu-files/images
3. Placed the configuration file en-test-eldp.json in:
   /library/www/html/js-menu/menu-files/imenu-defs
4. Edited menu.json to include a reference to our module.



Method 2: Integration via Local Content and Nextcloud
1. Uploaded the module directory to:
   /library/www/html/local-content
2. Cloned Nextcloud from the admin console and modified the relevant entries.
3. Updated the menu item definition accordingly.
